Mantenimiento Correctivo

¿Que es el Mantenimiento Correctivo? es la reparacion y el reemplazo de los componentes de hardware y software que presentan alguna falla o error, estos problemas se ven presentes con el pasar del tiempo y de su uso pueden provocar el recalentamiento de algunas piezas del ordenador o dar lentitud al rendimiento de un programa o aplicacion.

Tipos

Hay dos tipos de mantenimiento correctivo los cuales son el planificado que hace referencia al conocimiento temprano de que existe un problema pero no se ha solucionado y se prolonga por un tiempo indefinido.

El no planificado que se define por su mismo nombre, no se programa algun momento para ejecutar su solucion, sino mas bien se ejecuta en el mismo instante en que se presenta el error.

Pasos Para Realizar El Procedimiento

Naturalmente para cumplir cualquier tipo de mantennimiento se deben de cumplir ciertos pasos, en primicia se hace un diagnostico u revision del PC en la cual se encuentra el problema y se hace un analisis para hayar la resolucion del mismo.

Luego de saber con certeza el problema, se procederia a realizar el cambio o reparacion del componente dependiedo de la magnitud del daño que tenga este objeto, en dado caso si el componente esta muy afectado se debera reemplazarlo por otro nuevo.

Ya el componente sea reparado o sustituido se realizan pruebas en un software especial para comprobar la correcta Funcionalidad del mismo.

Herramientas

En toda clase de tareas se necesitan herramientas o utiles para poder cumplirlas, en este caso son las siguientes:

  1. Destornillador de estria u estrella
  2. Abrazaderas para cortar cables
  3. Pulseras Antiestaticas
  4. Pistola de aire comprimido
  5. Brocha
  6. Cepillos
  7. Guantes
