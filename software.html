Mantenimiento preventivo de software

Definición:

Se conoce al mantenimiento preventivo de software como el proceso por el cual se mejora y optimiza el software que se ha instalado en una computadora, este mantenimiento se realiza con el objetivo de prevenir posibles problemas que puedan llegar a surgir a medida que se utiliza el computador.

La principal razón por la que se realiza este tipo mantenimiento es el análisis de virus, dado que estos son programas nocivos para el computador, pues causan una inestabilidad en el sistema, bajas en el rendimiento del computador, pérdida de productividad, cortes en los sistemas y probables errores en el mismo, tales que no se pueden corregir de manera simple, sino mediante un formateo.

¿Que Es Software?

Pero que es realmente el ¿Software?, el software es un programa o un conjunto de programas, ordenes e instrucciones que permiten la ejecucion de tareas de cualquier tipo y es la parte logica del ordenador, su relacion con el hardware es del software envia ordenes al hardware para que esta las cumpla y de hacer posible el funcionamiento del ordenador.

Tipos De Software

Se puede definir que los tipos de software en sus multiples usos, comprende todos los programas y aplicaciones que son requeridas para la ejecucion general de todos los procesos del sistema, por tanto aplicaciones de inicio o distintos servicios que no solamente pueden existir en un ordenador, si no que tambien pueden se pueden usar en consolas, telefonos moviles o cualquier otro tipo de plataforma.

Software Por Funcionalidad

Se pueden clasificar en tres grandes grupos de acuerdo con las funciones que ofrecen. Especificamente hablando de software de aplicacion, software de programacion y software de sistema.

- Software De aplicacion

Dentro de esta categoria se puede hayar herramientas variadas que incluyen bases para el calculo, programas de emepresa o diseño, es una categoria muy extensa y abarca la ofimatica y editores de texto y otros enfocados al ambito recreativo o laboral.

- Software De Gestion

Se incluirian dentro de los denominados software de aplicacion como una herramienta que dan la facilidad en aspectos como la inventario o de gestion empresarial, desde contabilidad o de facturacion a la gestion de la nomina o de gestion de impuestos.

- Software De Programacion

La relevancia de este tipo de software radica en que es el software que genera el desarrollo de aplicaciones, sistemas, etc, por lo que es fundamental para el desarrollo de la informatica y los diferentes sistemas, dentro de esta categoria estan los editores de texto o compiladores.

- Software De Sistema

Es Fundamental para que el hardware funcione de manera correcta, las piezas del computador puede cumplir sus tareas gracias a este software, desde los controladores o drivers a los perifericos u otros elementos como lo son el teclado o el disco duro, un muy buen ejemplo de estos son los sistemas operativos Windows O Mac y su importancia se ve reflejada a su vez en que genera la conexion usuario y maquina.

Software Por Tipo De Licencia

Son Clave, ya que establecen los termino que permiten su utilidad correctamente, lo mas habitual es conseguirse con solo dos tipos, el software libre y el software de propietario, el primero es conocido por no disponer de un copyright y, por ende se puede usar sin tener que dar un pago por su o uso o derechos en el ambito legal lo mismo con su distribucion, su contraparte el software de propietario quedan en muchos limites como en su uso, distrubucion y copia

Software Por Tipo De Alojamiento

A la hora de la verdad, al momento de seleccionar seria el mas adecuado para una empresa u negocio podemos tambien valorar el tipo de tecnlogia por la cual se respaldan, un ejemplo de esto es On Premise y en la nube.

- Software On Premise

Esta basado en la instalacion y uso dentro de la propia empresa del programa que atraves de la correspondiente estructura interna, haciendo que la empresa se vea obligada a instalarlo en su servidor y hacerse cargo de los posibles riesgos venideros que eso pueda desarrollar.

- Software En La Nube

Este software permite su uso estando en la nube, es decir de otra forma usando un repositorio externo ala emrpesa al cual puede acceder de forma remota a cualquier hora desde cualquier lugar lo que facilita el aspecto de trabajo y manteniendo la productividad.
